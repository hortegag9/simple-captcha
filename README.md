# simple-captcha

Se va a realizar la demostración del funcionamiento de esta tecnología a través de un formulario web. Las herramientas y librerías utilizadas en este ejemplo fueron los siguientes:
-	STS 4
-	JDK 8
-	Apache Tomcat 9
-	Simple-captcha 1.2.1
-	Ognl 3.1.28
-	Strut2-core-2.1.6
-	Xwork 2.1.2
-	Javasisst 3.13.0
-	Freemarker 2.3.9
